Desarrollo de Aplicaciones Web (2022 - 2024)

Formación integral en el desarrollo, gestión y despliegue de aplicaciones. Se abarcan lenguajes como Java y JavaScript, junto con herramientas para el diseño de interfaces como Sass, Bootstrap y Tailwind CSS. En la parte de despliegue, se trabaja con Kubernetes, Docker y Apache, además de la administración de servidores y sistemas operativos Linux en entornos virtualizados. También se incluyen conocimientos en backend con PHP y bases de datos con MySQL.

React, Node.js, Express, API Rest - Udemy

Máster enfocado en el uso de componentes funcionales en React, el dominio de Hooks y la creación de Hooks personalizados. Se implementa React Router para la navegación SPA y Context API para la gestión de datos globales. También se exploran las novedades de React 18, como la concurrencia.

Next.JS - Udemy

Curso práctico para desarrollar una tienda online (E-Commerce) desde cero con Next.js, Strapi y Stripe. Se abordan la integración de frontend y backend de manera desacoplada, la gestión de datos con Strapi, la creación de endpoints personalizados y el uso de herramientas como Context API, JWT para autenticación y AWS S3 para almacenamiento multimedia.

Diseño UI + Figma - Udemy (Javier Cañas)

Curso orientado a la creación de aplicaciones móviles y páginas web atractivas y funcionales con Figma. Se trabajan conceptos como sistemas de diseño, diseño responsivo y principios fundamentales (Unidad, Gestalt, Jerarquía, etc.), además de aspectos clave como paletas de colores, iconografía, usabilidad, accesibilidad y tipografía. También se exploran metodologías como Atomic Design, el uso de grids y columnas, y la creación de Design Systems y UI Kits.

Hacking ético y Ciberseguridad (Santiago Hernández - actualmente) - Udemy

Curso práctico sobre técnicas avanzadas de ciberseguridad, incluyendo manipulación de tráfico de red en tiempo real y aplicación de Machine Learning en seguridad informática. Se utilizan herramientas como Kali Linux para la exploración de vulnerabilidades y protección de sistemas.
